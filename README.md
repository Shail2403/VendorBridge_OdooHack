# VendorBridge ERP

Procurement & Vendor Management ERP

## Features

- Vendor Management
- RFQ Creation
- Vendor Quotations
- Approval Workflow
- Purchase Orders
- Invoice Generation
- Activity Tracking
- AI Vendor Recommendation

## Roles

- Admin
- Procurement Officer
- Manager
- Vendor

## Workflow

Vendor Registration
→ RFQ Creation
→ Quotation Submission
→ Approval
→ Purchase Order
→ Invoice Generation
